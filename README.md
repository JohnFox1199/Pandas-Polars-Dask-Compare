Программа для сравнения производительности библиотек pandas, polars и dask при чтении и записи CSV файлов.

Программа измеряет:
- **Время чтения** CSV файла
- **Время записи** CSV файла  
- **Потребление памяти** при чтении и записи
- **Средние значения** по нескольким запускам

Все данные по измерениям выводятся в файл json.
